Introduction

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document It describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS has a bunch of tools to make the website one is making more presentible or complex, such as animations, font styles, color, etc.

What you should already know

This guide assumes you have the following knowledge:

  • Good working knowledge of HyperText Markup Language (HTML).
  • A general understanding of the Internet and the World Wide Web (WWW).
  • A willingness to be creative
How does CSS work

Let's use a example of CSS code right here:

p { color: red; }

This is a ruleset, (The term ruleset is often referred to as just rule.) Note the names of the individual parts:

Selector: This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled. To style a different element, you would need to change the selector. (In this case the "p" is the selctor used for this example.)

Declaration: A single rule. It specifies which of the element's properties you want to style. (In this case, it's "color: red;")

Properties: These are ways in which you can style an HTML element. (In this example, "color" is the property of "p" elements.) In CSS, you choose which properties you want to affect in the rule.

Property value: To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property. (For example, there are many "color" values in addition to "red".)

Uses for CSS

The main use for Cascading Style Sheets (CSS), is to make the appearence of a website a person is making unique and interesting, and even appealing to look at while someone is browsing the website.

The HTML elements can be thought of as boxes, sitting on top of other boxes. These boxes can be changed and shaped and repositioned to the designer's desires, with the rulesets used in CSS. The types of changes one can make include the following:

  • Width/Height: The basic size of the object. You can change it to be large or small, or however you wish for it to be. (e.g. li { width: 20px; height: 20px;})
  • Background-Color: This changes how the color of a object's background color, making the box not only appear, but also be the desired color. Good for making aesthetic objects that are to decorate the website with. (e.g. li { Background-Color: #4A79A5} (This would change the background color to be the color Denim))
  • Color: The color of the contents within the object, which is usually text. (e.g. li { Color: #A074F1} (This would change the text color to be the color Portage))
  • Text-Shadow: The ruleset that allows text of a object to have a shadow of some kind within said object. (e.g. li { text-shadow: #FC0 1px 0 10px;} (This would change the text shadow to appear as having a soft glow around it.))
  • Display: Changes how the object is displayed. (e.g. li { display: flex;})
  • Border The border of a object. It can ether be invisable or visable. (e.g. li { border: thick double #32a1ce;})
